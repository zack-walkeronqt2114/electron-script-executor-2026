## Overview

Electron is a Roblox script executor built for Windows desktop users who want to run Lua scripts from a local app. It pairs a compact interface with an integrated script hub, making it easier to load, stage, and execute scripts without a complicated setup process.

The project focuses on everyday scripting workflows, with features such as one-click injection, batch execution, and auto-update support. It is intended for users who want a Roblox script executor with a large included script library, offline desktop operation, and an interface that remains straightforward to use on Windows 10 and Windows 11.
